# Guess The Number Game

A simple browser-based game where the user tries to guess a randomly generated number.

## Technologies Used
- HTML5
- CSS3
- JavaScript

##  How to Play
1. Enter a number within the given range.
2. Click the guess button.
3. The game will give feedback:
   - Too high
   - Too low
   - Correct 
4. Keep trying until you guess the correct number.

## Features
- Random number generation
- User input validation
- Instant feedback
- Clean and simple UI

##  Purpose
This project was created to practice JavaScript fundamentals and basic game logic.

- Add difficulty levels
